## 👨‍💻 Sobre
 
Desenvolvedor backend em formação com foco em **Java e Spring Boot**. Construo APIs REST do zero — modelagem de domínio, persistência com JPA/Hibernate, segurança com Spring Security — sem depender de tutoriais ou scaffolding pronto.
 
Paralelamente, faço **Iniciação Científica em IA** no Projeto Zélia (UNIJORGE), desenvolvendo agentes conversacionais RAG + ReAct com LangChain, LangGraph e Google Gemini. Membro fundador da **LIAU** (Liga Acadêmica em Inteligência Artificial).
 
> Procuro minha primeira oportunidade como **estagiário ou desenvolvedor júnior backend** — presencial em Salvador/BA ou remoto.
